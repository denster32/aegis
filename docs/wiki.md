# Project Wiki

AutoWiki-style generation into `docs/wiki/`.

```bash
aegis wiki generate
aegis wiki refresh
aegis install-wiki-refresh    # GH workflow / automation helper
```

## Typical pages

- `Home.md` — overview
- `Architecture.md` / `Modules.md` — structure
- `Commands.md` — CLI map
- `Conventions.md` — project norms
- `_Sidebar.md` — navigation

Hand-edit after generate; `refresh` reconciles with current tree.
