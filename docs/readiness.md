# Agent Readiness (L1–L5)

Factory-inspired **local** readiness model: how prepared this repo is for long autonomous agent work.

```bash
aegis readiness           # default v2 pillars
aegis readiness --json
```

## Levels (summary)

| Level | Theme | Examples |
|-------|--------|----------|
| L1 | Basics | Git, manifest, build entrypoint |
| L2 | Quality | Tests, fmt/lint, CI |
| L3 | Memory | `.aegis/`, lessons, skills |
| L4 | Process | Missions, QA, review hooks |
| L5 | Autonomy | Automations, dream, wiki refresh |

Levels unlock around an ~80% criteria pass rate per pillar (implementation detail may evolve).

Higher readiness → better long Missions and fewer dead-ends.

Related: [factory.md](./factory.md), [missions.md](./missions.md).
